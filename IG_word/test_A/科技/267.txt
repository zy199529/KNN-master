<title>黄粱美梦 靠WinXP退休增加Win8市场份额|微软|WinXP|Win8_笔记本_新浪科技_新浪网</title> 早在一年前，微软就已宣布将于2014年4月8日停止对Windows XP系统以及Office 2003提供官方技术支持，并通过多种方式不断提醒大家升级Windows系统。如今这个日期越来越近，原本是微软正常的Windows产品生命周期，为何引发业界如此高度关注？微软期望借此契机推动新一代操作系统的如意算盘能否如期兑现？ Windows XP零售版于2001年10月25日正式上市，到今天已经走过了十二个年头，该系统做了大量功能改进，以较低的硬件需求和友好的界面体验迅速获得用户亲睐，成为市占率较高的Windows版本并且延续至今。但现在，WinXP的巨大成功已经成为微软在移动互联网时代推广自己全新跨平台操作系统以及搭建全新生态系统的最大障碍，Windows 8发布一年多以来销量一直裹足不前，令微软非常头痛。如今，Windows XP生命周期已尽，按计划停止官方技术支持原本是“顺理成章”的事情，但微软正好可以借此千载难逢的好机会煽煽风加把火，试图将数量众多的Windows XP用户招至Win8麾下，既可以凭借昂贵的Win8系统价格和OEM设备授权费用获得不菲的利润，更能有效地扩大和稳固自己的跨平台生态系统，以期增加与苹果和Goolge等巨头的竞争力。 据统计，目前全球依然有超过25%的用户还在使用Windows XP操作系统。2013年11月的《微软安全报告》中提到，中国内地Windows XP系统用户高达57.8%，近期的统计数据表明，国内Windows XP的用户比例仍在50%上下，落实到具体数量，国内使用WinXP的用户数量不在2亿之下。2亿这个用户数量对于任何公司都是一块充满诱惑的肥肉，但是，愿望是美好的，现实是残酷的，想仅凭停止WinXP官方技术支持就期望大部分XP用户转化成Win8用户，只是看上去很美的黄粱美梦。 脱保XP不安全？有免费“篱笆墙” 从去年开始，微软就在一直提醒用户，一旦Windows XP失去官方技术支持，系统安全隐患将迅速增大，XP用户岌岌可危。微软官方网页醒目位置提示：“2014年4月8日之后，我们不再提供 Windows XP 技术帮助，包括帮助保护你的电脑的自动更新。Microsoft 还将在此日期之后停止为 Windows XP 提供 Microsoft Security Essentials 下载服务。”而坊间也曾经流传着所谓“4月8日以后WinXP电脑将在10分钟内被病毒感染”的危言耸听以及ATM机飞钞票的夸张视频(注：其实银行的ATM网络系统是内部封闭系统，不与外界网络相连接，安全性相对较高)，连微软自己也出来澄清：“4月8日之后Windows XP系统仍然可以继续使用。但是这款长达13年的产品已经不能满足互联网时代的需求，不足以应对层出不穷且变化多端的网络安全威胁。” 的确，在互联网技术以及互联网金融应用不断发展的今天，失去官方技术支持的Windows XP用户面临木马病毒感染、敏感信息泄露、电脑无法正常运行的风险会增加。国内的安全服务厂商自然不会看着2亿的用户市场不闻不问，Windows XP即将停止官方技术支持的消息一出，就有不少系统网络安全公司先后宣布将向所有个人及企业用户提供永久免费的XP系统补丁支持服务、安全保护以及多套安全解决方案等等。 对于安全服务厂商而言，2亿多的WinXP用户市场自然是兵家必争之地，不管是单独提供后XP时代的安全服务还是打上微软这个“官方合作”的金字招牌，既能抢夺用户又能扩大影响力，一圈圈篱笆们先围住羔羊。至于4月8日以后是否能够有效抵挡黑客们的攻击，还需时间检验。 对于微软而言，这道“篱笆墙”也有双重防护作用：一旦真的爆发大规模的黑客攻击导致重大安全事故，微软大可以拍拍屁股说：早就警告过你们WinXP停止官方支持之后会有严重的安全隐患，事实证明不升级系统和设备果然后患无穷；如果一切风平浪静，或者偶有些小打小闹被“篱笆”们迅速摆平，微软也可以欣慰表示已经与合作伙伴们尽到“业界良心”。 如果说之前关于XP脱保后系统安全威胁提醒的确让一些电脑小白有些恐慌，现在这些第三方免费的安全服务似乎又给WinXP的死忠们吃下了定心丸。这么多厂商都争先恐后地提供免费的WinXP安全服务，何必着急升级系统呢？ XP落后用户守旧？企业单位应用迁移难 为了推动WinXP用户升级，有些论调一直在批评WinXP用户“守旧，不能接受新鲜事物，跟不上时代的发展”云云，其实事实并非完全如此。有业内人士表示，在部分部委和大型国企，XP系统应用比例最低超过60%，最高的甚至接近95%。由于很多政府机关和企业的信息系统应用、财务系统都是基于Windows XP系统环境下进行开发的，升级操作系统之后难以兼容，无论是应用迁移还是再次开发都需要付出大量的人力、金钱和时间成本。而且不少企业之间的数据交换也需要相同系统兼容，如果升级速度参差不齐也会造成巨大的工作障碍。这些企业和单位的专业应用从WinXP系统向Windows 8系统迁移难以在短时间内完成，有分析称保守估计这一过程将持续1-5年。 Win8优秀？用户不买账 对于微软极力推崇的新一代跨平台的操作系统Windows 8，个人和企业用户的接受度也相当有限。微软于2012年10月底推出结合触控与移动特性的全新一代操作系统Windows 8，试图将平板电脑的直观易用性和桌面电脑的的兼容性及功能进行整合，推崇Win8设备胜任工作生活双重需求的特性，但最终却处境尴尬。不少消费者对Windows 8怨声载道，在非触控设备使用Modern界面和应用实在别扭，而缺少开始菜单的传统桌面也让用户无所适从。  微软寄希望于Win8能够迅速推广，搭建自己的移动互联网时代的全新生态系统，PC厂商也推出了品牌多样种类丰富的各种Win8设备，期盼能借助Windows 8的力量重振整个行业，同时应对向移动产品转型所带来的挑战。可惜市场反映并不如意，到现在Win8的全球市场占有率仍在10%徘徊。 告别XP？成本巨大 就算在XP退休的强势宣传攻势下，有部分用户和企业单位开始考虑升级Windows系统和购买新设备，他们需要付出的成本也是巨大的。 微软在其官方网站给出了告别XP的两个途径： 1、升级系统微软建议：“只有极少数的旧版计算机将能够运行 Windows 8.1，它是最新版本的 Windows。建议你下载并运行 Windows 升级助手以查看你的电脑是否满足 Windows 8.1 的系统要求，如果你的电脑满足要求，则可以按照教程中的步骤升级你的电脑。” 2、购买新电脑微软建议：“如果你当前的电脑无法运行 Windows 8.1，则可能该考虑购买一台新电脑了。浏览我们提供的全新精选电脑。 它们的功能比以往更强大、更轻盈、更时尚，并且价格适中，比 10 年前电脑的价格便宜很多。”这句“便宜很多”引发了不少网友的吐槽。 目前Windows 8操作系统的优惠价格为888元(Windows 8推出时曾有优惠价98元)，搭载Windows 8系统的设备最低也在3000元左右，对于个人用户而言这笔开支是需要考虑和掂量的。而对于拥有大量WinXP电脑设备的的机构和企业单位来说，这些WinXP电脑大多年代久远，配置不高，满足不了单纯升级新版Windows系统的要求，想要升级系统就必须重新购买电脑设备。就算是电脑配置满足新版Windows的需求，批量采购升级系统的费用也不低。所以，对于企事业单位而言，无论是升级Windows系统还是购买预装新系统的电脑设备，都需要大笔的经费支出。 延续XP服务？有偿提供 对于少数暂时不想升级XP系统却依然想要得到微软官方支持的用户，微软也提供了一些有偿服务。 爱尔兰政府与微软签订了一项费用为450万美元的Windows XP延长支持协议。根据协议，微软将在2014年4月8日之后继续向爱尔兰政府机构的计算机提供12个月的补丁。爱尔兰政府发言人称：“我们与微软签署了一份备忘录，他们会于今年4月至明年4月之间的12个月里优先向我们提供安全补丁，”据悉，这份备忘录覆盖的爱尔兰政府部门包括卫生部、司法部、环保部和教育部。据消息人士透露，最开始微软提出提供12个月的技术支持需要至少1940万美元资金支持，但经过爱尔兰政府的讨价还价，最终价格定格在450万美元。 外媒消息称，ATM机制造商NCR声称全球约有三分之二的ATM机目前仍在运行Windows XP，即150万台左右的ATM机将无法在Windows XP退休前完成升级Win7、Win8或Win8.1。如果要选择延长支持Windows XP服务，费用非常昂贵，有分析人士透露“每家英国主要银行延长支持Windows XP并升级新平台的费用在5000万到6000万英镑(约合1亿美元)之间。 综上所述，虽然Windows XP停止官方服务其实原本就是微软Windows产品正常的生命周期结束，但因为Windows XP系统的用户量过大，如此巨大的市场契机自然引发了IT业界的广泛关注。 微软急于在移动互联网时代快速推广自己的跨平台新一代操作系统Windows 8，渴望搭建自己的全新生态系统，不断大打“安全”这张牌想要吸引WinXP用户转向Win8。但对于Windows XP的忠实用户而言，Windows XP已经是使用多年的工作和生活伙伴，不仅习惯力量巨大，很多信息财务等专用系统也难以迁移，旧系统用起来顺手，新系统换起来棘手，而且，无论是升级系统还是购买新设备，都需要花钱，而且这笔开销并不算小，综合考虑，很多个人消费者和企业单位自然会有“能不换就不换”的想法。 即使微软和媒体都不停地在提醒Windows XP停止支持将带来非常大的安全隐患，市场上也提供了琳琅满目的Windows 8设备，但是，想要让Windows XP的死忠们心甘情愿地掏钱升级系统或者更换Windows设备，还需要更多从用户角度出发的贴心服务和优惠措施。商家追求利润最大化，用户更看重投入产出比。 微软说服用户离开WinXP的推力不足，Win8本身对用户的吸引力又不够，想要WinXP用户心甘情愿进入Win8阵营，这如意算盘远非安全威胁和守旧讥讽就能奏效的。
